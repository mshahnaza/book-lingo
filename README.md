# BookLingo

## 1. Описание проекта

**BookLingo** — веб-приложение для изучения иностранных языков через чтение книг. Пользователи могут загружать книги в форматах PDF и TXT, читать их постранично, переводить незнакомые слова, сохранять их в персональный словарь и создавать флеш-карточки для повторения с использованием алгоритма интервального повторения SM-2.

## 2. Технологический стек

### Backend
- **Фреймворк**: Spring Boot 3.2.4
- **Язык**: Java 17
- **База данных**: H2 (разработка) / PostgreSQL (продакшн)
- **ORM**: Spring Data JPA + Hibernate
- **Безопасность**: Spring Security + JWT
- **Маппинг объектов**: MapStruct 1.5.5 + Lombok
- **Обработка PDF**: Apache PDFBox 2.0.30
- **Перевод**: LibreTranslate / Google Translate API

### Frontend
- **Фреймворк**: React 18
- **Сборка**: Vite
- **Стилизация**: CSS-in-JS (инлайн стили)
- **HTTP-клиент**: Fetch API

## 3. Архитектура приложения

### 3.1 Структура Backend

Приложение следует классической многослойной архитектуре:

```
Controllers → Services → Repositories → Entities
```

**Основные сущности:**
- `User` — пользователь системы (реализует Spring Security UserDetails)
- `Book` — загруженная книга с метаданными
- `Page` — страница книги с извлеченным текстом
- `Word` — сохраненное слово с переводом
- `Flashcard` — карточка для повторения
- `LearningSession` — запись о сессии обучения

### 3.2 Структура Frontend


## 4. Реализованный функционал

### 4.1 Аутентификация и авторизация
- Регистрация нового пользователя
- Вход в систему с получением JWT токена
- Автоматическое обновление refresh токена
- Защита всех API эндпоинтов

### 4.2 Управление книгами
- Загрузка книг в форматах PDF и TXT
- Отображение списка книг пользователя
- Удаление книг
- Отслеживание прогресса чтения (текущая страница)

- Запоминание последней прочитанной книги

### 4.3 Чтение книг
- Два режима чтения: постраничный и скроллинг
- Навигация по страницам
- Выделение текста для перевода
- Закладки (сохраненные слова на странице)

### 4.4 Перевод и словарь
- Перевод выделенного текста
- **Поддержка 12 языков**: английский, русский, испанский, французский, немецкий, итальянский, португальский, китайский, японский, корейский, арабский, хинди
- Выбор исходного и целевого языка
- Сохранение слов в персональный словарь
- Группировка слов по книгам
- Поиск по словарю

### 4.5 Флеш-карточки
- Создание карточек из сохраненных слов
- Создание пользовательских карточек
- Алгоритм интервального повторения SM-2
- Статистика по карточкам (всего, к повторению сегодня, выучено)
- Отслеживание качества ответов (0-5)

### 4.6 Статистика
- Количество прочитанных книг
- Количество сохраненных слов
- Количество созданных карточек
- Карточки к повторению сегодня

## 6. API эндпоинты

### Аутентификация
- `POST /api/auth/register` — регистрация
- `POST /api/auth/login` — вход
- `POST /api/auth/refresh` — обновление токена

### Книги
- `POST /api/books/upload` — загрузка книги
- `GET /api/books` — список книг пользователя
- `GET /api/books/{id}` — получение книги по ID
- `DELETE /api/books/{id}` — удаление книги
- `PUT /api/books/{id}/position` — обновление позиции чтения

### Страницы
- `GET /api/pages/book/{bookId}` — получение страниц книги

### Слова
- `POST /api/words` — сохранение слова
- `GET /api/words` — список слов пользователя
- `DELETE /api/words/{id}` — удаление слова

### Флеш-карточки
- `POST /api/flashcards` — создание карточки
- `GET /api/flashcards` — список карточек
- `GET /api/flashcards/due` — карточки к повторению
- `PUT /api/flashcards/{id}/review` — отметка о повторении
- `DELETE /api/flashcards/{id}` — удаление карточки

### Перевод
- `POST /api/translate` — перевод текста

## 7. Алгоритм SM-2

Реализован алгоритм SuperMemo 2 для интервального повторения:

**Шкала качества (0-5):**
- 0-2: Неверный ответ → сброс интервала на 1 день
- 3-5: Верный ответ → увеличение интервала

**Расчет интервала:**
- 1-й верный ответ: 1 день
- 2-й верный ответ: 6 дней
- Последующие: `интервал × коэффициент легкости`

**Коэффициент легкости:**
- Начальное значение: 2.5
- Формула корректировки: `EF' = EF + (0.1 - (5-q) × (0.08 + (5-q) × 0.02))`
- Минимальное значение: 1.3

## 8. Безопасность

- JWT токены с коротким временем жизни (15 минут)
- Refresh токены для обновления (7 дней)
- Изоляция данных пользователей (все запросы фильтруются по userId)
- Хранение файлов в изолированных директориях пользователей
- Защита от CSRF (stateless аутентификация)

## 9. Выполненные улучшения

В ходе разработки были реализованы следующие улучшения:

1. **Исправление индикатора страницы в режиме скроллинга** — использование ref для отслеживания текущей страницы без стale closure
2. **Редизайн интерфейса** — переход на минималистичную темную тему с нижней навигацией
3. **Мультиязычный перевод** — добавлена поддержка 12 языков с выбором в popup
4. **Тост-уведомления** — замена стандартных alert() на стилизованные уведомления
5. **Сохранение позиции чтения** — автоматическое сохранение с debounce (1 сек)
6. **Отслеживание последней книги** — сортировка по lastReadAt для "Продолжить чтение"
7. **Исправление подсчета карточек** — корректное сравнение дат для due today
